User Information System

■Description

This Python project collects user data such as name, age, weight, and favorite color. It processes the information using basic programming concepts and displays meaningful output.

■ Features

User input collection

Type casting (int, float)

String manipulation

Conditional logic

Basic calculations

Requirements

Python 3.x

How to Run

1. Open terminal

2. Navigate to project folder 3. Run: python your_file_name.py
